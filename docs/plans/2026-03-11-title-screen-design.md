# タイトル画面 Design

## Goal

`#1 タイトル画面の作成` を、既存の静的クイズ構成を保ったまま最小差分で実現する。

## Decision

- 初回表示はタイトル画面にする
- タイトル画面とクイズ画面は同一 HTML 内の view 切り替えで実装する
- 主導線は `クイズをはじめる` ボタン 1 つに絞る
- 将来の `ずかん` 導線は、今回は `じゅんびちゅう` の無効ボタンとして配置する
- クイズ完了後の `ぜんぶ できたね！` と `もう いっかい` は既存フローを維持する

## Why

- GitHub Pages で配信する単一ページ構成を維持でき、ルーティングや複数 HTML 管理を増やさずに済む
- タイトル画面を別ページに分離しなくても、3歳児向けの単純な導線を実現できる
- `ずかん` の枠を先に UI 上で確保しておくと、将来の機能追加時に情報設計を崩しにくい

## Scope

- `index.html` にタイトル画面セクションを追加する
- `styles.css` にタイトル画面用レイアウトと無効導線のスタイルを追加する
- `js/app.js` に view 状態管理と開始導線のイベントを追加する
- DOM テストで初回表示と導線切り替えを固定する
- README にタイトル画面の説明を追記する

## Non-Goals

- `ずかん` 機能そのものの実装
- ルーティング導入や複数ページ化
- クイズロジックや出題ルールの変更
